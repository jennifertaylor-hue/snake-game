# Competitive Snake Game

## Overview
The Competitive Snake Game is a modern twist on the classic snake game, where players can compete against each other in real time. The objective is to eat as much food as possible to grow the snake while avoiding collisions with walls and other players.

## Features
- **Real-time Multiplayer:** Compete against friends or random players online.
- **Power-ups:** Collect power-ups to gain special abilities for a limited time.
- **Customizable Snakes:** Choose from different snake designs and colors.
- **Leaderboard:** Track your scores and compare them with players around the world.

## Getting Started
### Prerequisites
- A web browser with JavaScript enabled.

### How to Play
1. Visit the game URL.
2. Join or create a game room.
3. Use the arrow keys to control your snake.
4. Eat the food to grow your snake and avoid crashing into walls or other snakes.
